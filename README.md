# 🌍 NGO Website – HopeReach

A modern NGO website built using **React + Vite** to showcase initiatives, encourage volunteering, and spread awareness.

## 🚀 Live Demo
🔗 Live Website: https://melodic-crostata-e44d38.netlify.app/

## 📌 Features

- Responsive modern UI
- Hero section with NGO mission
- Volunteer registration page
- Gallery section
- Contact page
- Social media integration
- Splash screen animation
- React Router navigation

## 🛠️ Tech Stack

- React.js
- Vite
- React Router
- Lucide Icons
- CSS3
